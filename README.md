## ğŸš€ Getting Started

### Prerequisites

- Node.js >= 18
- React Native CLI
- Android Studio (for Android)
- Xcode (for iOS, macOS only)
- JDK 17

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Taimooriqbal125/prayer_app.git
   cd prayer_app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **iOS only - Install CocoaPods**
   ```bash
   bundle install
   bundle exec pod install
   ```

### Running the App

**Start Metro Bundler**
```bash
npm start
```

**Run on Android**
```bash
npm run android
```

**Run on iOS**
```bash
npm run ios
```
